# Elite Home Craft

A professional, responsive website for Elite Home Craft - premium home repair and renovation services.

## Features

- **Hero Section** - Eye-catching hero with service highlights and stats
- **Services Grid** - 6 key service offerings with icons and descriptions
- **Portfolio Gallery** - Before/After image sliders showcasing completed work
- **Testimonials Carousel** - Client reviews with auto-advance feature
- **About Section** - Company information and credentials
- **Service Areas** - Interactive map and list of coverage areas
- **Contact Form** - Professional contact form with validation
- **Fully Responsive** - Works perfectly on all devices

## Technologies Used

- HTML5
- CSS3 (Custom properties, Grid, Flexbox)
- Vanilla JavaScript (ES6+)
- Google Fonts (Playfair Display + Inter)

## Live Demo

Deployed on Vercel

## Local Development

Simply open `index.html` in your browser, or run:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000`

## Customization

To customize for your business:

1. Update contact information in `index.html` (phone, email, address)
2. Replace service descriptions in the Services section
3. Add your own images by replacing the gradient backgrounds
4. Update color scheme in `styles.css` (CSS variables at the top)
5. Modify testimonials with real client reviews
